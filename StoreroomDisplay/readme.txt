通信协议：

0x55 ID  CMD   data  LRC  #


ID：     呼叫器工位号对应ID为 0x01~9，仓库显示器ID为 0x01

CMD：   'c': 呼叫器呼叫     命令
        'g': 收到回复       命令
        'u': 更新物料信息   命令
        'e': 校验错误

size：   data长度（字节）

data：   数据
        1. 更新的物料信息（char*，以 ',' 分隔不同物料）
        2. 呼叫的物料编号（0x01 ~ 0x09）
        3. 其他默认为0x00

LRC ：   校验位
